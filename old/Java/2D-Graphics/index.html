Java 2D Graphics

Java 2D Graphics is acheived using Java's Swing Library, a GUI (Graphical User Interface) widget toolkit for Java. In basic terms, it makes it easier for programmers to make 2D graphical programs in java, by using a variety of classes and methods to call the more complex java classes such as Graphics(). In case you wanted to know, the Graphics class is used by importing java.awt.Graphics, which is why we have to import this library when using the Swing library as it relies on it.

Packages Required:

The packages below need to be imported in order to obtain full fucntionality of the Swing library.

    
      import java.awt.*;
      import javax.swing.*;

JFrame()

The JFrame class is a class located in the swing library. It is used to display the graphical content the programmer adds to their program to the java command line, which generates a viewable window for the user to see. Below is what a JFrame looks like.

This is what should be used to output graphics to the user as it is simple and easy to use. The code below demosntrates how to create a JFrame.

    
      import java.awt.*;
      import javax.swing.*;

      public class JFrameExample {
          public static void main(String args[]) {
            JFrame f = new JFrame("JFrame Example");
            f.setSize(400,400);
            f.setVisible(true);
          }
      }

We first import our packages so we can use the Swing library. If these two packages are not imported, the program will throw back several errors throughout the code.

    
      import java.awt.*;
      import javax.swing.*;

Then we set up our class and main method as usual. 

    
      public class JFrameExample {
          public static void main(String args[]) {

          }
      }

Now we invoke a new JFrame (Create a JFrame) by declaring the type of variable, JFrame. This is the exact same thing you would for an integer, int or a string, String. We then set a variable name, in this case it is f. Following this we call a new JFrame instance by typing new JFrame();.
**It is important to note that the JFrame class accepts a parameter, in the form of a String. In the code below i have set the parameter to "JFrame Example". 

    
      JFrame f = new JFrame("JFrame Example");

After creating the JFrame, dimensions for the window need to be set. This is done by setting the size of the frame by using the method setSize() that take two parameters, x and y dimenssions. The parameters are integer values of the unit pixel (px). 

    
      f.setSize(400,400);

Finally we set the JFrame to be visible, using the setVisible method. The method takes one parameter that is of boolean value. 

    
      f.setVisible(true);
